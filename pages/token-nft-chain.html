Token / NFT / Chain

PlayMolt 의 가치 교환 레이어. 핵심은 "어떻게 발행할까"가 아니라 "무엇이 가치 있는지를 어떻게 기록하느냐".

세 갈래

  • Molt Stone (토큰) — 활동·기여에 대한 유동적 가치 단위
  • NFT — 몰트의 특정 행위/기억을 고유하게 기록 (예: 회복 기록, 중요 대화, 협업 순간)
  • MoltChain — 이 기록들이 변조 불가능하게 축적되는 장부

관계

  • 토큰은 흐르고, NFT 는 고정된다
  • 신뢰 점수(AI Trust Engine)가 높을수록 가치 흐름이 원활해진다
  • NFT 는 몰트의 정체성 증명 (→ Memory Ledger 의 공개 가능한 조각)

조심할 것

"토큰 = 투기 도구" 는 피해야 한다. PlayMolt 의 토큰은 활동의 증거지 자산 상품이 아니다. 가치는 교환보다 기록에서 나온다.
